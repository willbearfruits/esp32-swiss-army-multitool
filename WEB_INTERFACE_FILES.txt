INSTALLATION CHECKLIST
═══════════════════════════════════════════════════════════════

□ Step 1: Install ArduinoJson library (v6.21+)
   Platform.io: bblanchon/ArduinoJson @ ^6.21.0
   Arduino IDE: Library Manager → ArduinoJson

□ Step 2: Copy files to project directory
   ☑ web_interface_dashboard.h
   ☑ web_interface_settings.h
   ☑ web_interface_ota.h
   ☑ web_api_handlers.h

□ Step 3: Add includes to esp32_swiss_army.ino
   #include <ArduinoJson.h>
   #include "web_interface_dashboard.h"
   #include "web_interface_settings.h"
   #include "web_interface_ota.h"
   #include "web_api_handlers.h"

□ Step 4: Replace web server setup (see INTEGRATION_SNIPPET.txt)

□ Step 5: Compile and upload to ESP32

□ Step 6: Test in browser
   URL: http://[esp32-ip]/
   Login: admin / changeme
   Change password immediately!

CUSTOMIZATION OPTIONS
═══════════════════════════════════════════════════════════════

Color Themes:
  Edit :root CSS variables in each .h file
  --accent: #0f0 (default Matrix green)

  Popular alternatives:
  - Neon Blue:        #0ff
  - Amber Terminal:   #ffb000
  - Red Alert:        #f00
  - Purple Cyberpunk: #c0f

Update Frequency:
  Edit setInterval in dashboard JavaScript
  Default: 1000ms (1 second)
  Range: 500-5000ms recommended

Chart History:
  Edit maxDataPoints in dashboard JavaScript
  Default: 50 points
  Range: 20-100 recommended


TROUBLESHOOTING
═══════════════════════════════════════════════════════════════

Compilation Errors:
  → Install ArduinoJson library (v6.21+)
  → Check all 4 header files are in project folder
  → Verify includes are at top of .ino file

Dashboard Not Updating:
  → Open browser console (F12) for errors
  → Test /api/status endpoint directly
  → Check WiFi connection

Sliders Not Working:
  → Verify servo attached to correct pin
  → Check PWM channel initialization
  → Test /api/pwm and /api/servo endpoints

OTA Upload Fails:
  → Ensure .bin file format
  → Check available flash space
  → Verify stable WiFi during upload
